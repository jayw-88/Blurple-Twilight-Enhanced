# Blurple-Twilight-Enhanced

A theme for Vencord inspired by the Blurple Twilight Nitro theme, implemented with the EmojiReplace (Apple) theme.

## How To Install

Download the "bte-theme.css" file from the repo and place the file into the Vencord "themes" folder. Click the "Loading Missing Themes" button, and the theme should show up. Enable it and you're all set.
